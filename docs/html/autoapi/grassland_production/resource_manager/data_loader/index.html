grassland_production.resource_manager.data_loader

The Data Loader module provides a set of classes and methods to facilitate the retrieval of various agricultural data. The data is retrieved from a DataManager instance which manages the underlying database connections and queries.

Classes:

Loader: Provides methods to retrieve data from a DataManager instance.

class grassland_production.resource_manager.data_loader.Loader[source]

The Loader class is designed to facilitate the retrieval of various agricultural data related to grassland and livestock management from a DataManager instance. This class primarily focuses on providing easy access to different sets of data related to grassland fertilization, livestock areas, soil groups, and livestock numbers.

grassland_fertilization_by_system()[source]

Retrieves data on grassland fertilization application rates segmented by various systems and grassland types.

This method accesses the DataManager’s ‘get_grassland_fertilization_by_system’ function to fetch data related to the application of fertilizers in different grassland management systems.

Returns:

A pandas DataFrame containing detailed information about

grassland fertilization rates segmented by system and grassland type.

DataFrame
